## Setup Arduino
Clone
```
git clone https://github.com/kiyoto1022/arduino_time_recorder.git
```

Open Arduino File
```
ArduinoIDE > File > Open > arduino_time_recorder.ino
```

Writing Arduino Board
```
ArduinoIDE > Sketch > Upload
```
